Overview


Click the tabs to learn about electric field concepts, how to calculate them, and how to tell the application (Trinket) to calculate it for you.

Note that Trinket supports two versions of Python, which can be specified at the top of the script. For python 2, add #!/bin/python2 and for python 3, add #!/bin/python3. Also note that Trinket does not support all python modules. For a list of the currently available modules, click HERE.

Trinket

Each of the tabs feature a Trinket to execute the code. To save specific Trinkets for your own reference, complete the following steps:

  • Go to the Trinket sign-up page.
  • Create an account.
  • On the embedded Trinket, click the image link on the top right to log in using your account email and password.
  • Click the Remix button to save a copy of the code to your Trinket account.
  • Save the Trinkets with a unique name for future reference.
